# Daemons & Skills Catalog

This repo contains general-purpose example daemon files and agent skills for Charlie.

To use these files, copy them into your repo at:

- `skills/<name>/**` -> `.agents/skills/<name>/**`
- `daemons/<name>/**` -> `.agents/daemons/<name>/**`

Most example daemons use placeholders such as `<issue-id-pattern>`, `<daemon-signature>`, and `<candidate-root-globs>` so you can adapt them to your repo, tracker, and review workflow with minimal changes.
